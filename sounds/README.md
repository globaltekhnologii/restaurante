# Archivos de Sonido para Notificaciones

Los archivos de sonido deben colocarse en la carpeta `sounds/`:

## Archivos Requeridos:

1. **new_order.mp3** - Sonido de campana para nuevos pedidos
2. **order_ready.mp3** - Sonido de "ding" para pedidos listos
3. **alert.mp3** - Sonido de alerta general

## Dónde Obtener Sonidos:

Puedes descargar sonidos gratuitos de:
- https://mixkit.co/free-sound-effects/notification/
- https://freesound.org/
- https://notificationsounds.com/

## Características Recomendadas:

- **Duración:** 1-2 segundos máximo
- **Formato:** MP3
- **Volumen:** Moderado (se ajusta en código a 70%)
- **Tipo:** Sonidos cortos y claros

## Alternativa Temporal:

Si no tienes los archivos de sonido, el sistema funcionará sin ellos.
Las notificaciones visuales seguirán apareciendo normalmente.
